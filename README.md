# üöÄ Android Task - Word Count App üì±

**Android Task** is a modern Android application that **fetches words from a website, counts
occurrences, and displays them in a sorted list**.  
Built using **Jetpack Compose, Clean Architecture, MVVM, Flow, Room DB, Retrofit**, and follows *
*Google's best practices**.

---

## üéØ **Features**

‚úî Fetches **word count from the Bazzarry website**  
‚úî **Offline Support** using **Room Database**  
‚úî **Search functionality** for filtering words  
‚úî **Sorting by count (Ascending & Descending)**  
‚úî **Jetpack Compose UI** for a modern, smooth experience  
‚úî **Lifecycle-aware with StateFlow**  
‚úî **Handles device rotation and maintains UI state**
